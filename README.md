## Compose - Comprender los mecanismos para correr aplicaciones Docker multi-container

1.	Configure un servidor FTP usando docker compose
2.	Usando Docker Compose empaquete la aplicación desarrollada en la práctica de API REST
3.	Demuestre el uso de docker compose para otra aplicación de su interés. Puede tomar uno de los ejemplos descritos aquí:
https://docs.docker.com/compose/samples-for-compose/ o usar otras fuentes.
