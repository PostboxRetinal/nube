Apache + Azure + AWS

Host Apache en Azure consumiendo API REST en AWS EC2

Esta página sirve como frontend estático en Apache de Azure y reenvía las solicitudes hacia la API de Elysia ejecutándose sobre Bun en AWS EC2.

Azure VM

Apache sirve este frontend y actúa como proxy inverso hacia la API alojada en AWS.

AWS EC2

La API REST de Elysia responde en /api/data y devuelve JSON que luego se muestra aquí.

Estado: esperando acción 
Haz clic en el botón para ejecutar la API.

Si Apache está configurado con proxy inverso, la URL utilizada es /api/data. Si no, ajusta el proxy para que apunte a http://<AWS_PUBLIC_IP>:3000/api/data.
